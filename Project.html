My Academic Project

IOT Mining Tracking & Worker Safty Helmet

Aim of the project

   The main aim of this project is to provide safety and security to the Mining workers who are working in the underground Mines by monitoring the weather conditions and location of the worker.

Components used in the project

Hardware Components

  • Node MCU ESP8266 Microcontroller with Inbuilt Wifi module
  • GPS Module
  • MQ2 Gas Sensor
  • Temperature & Humidity Sensor(DHT11)

Software Components

  • Arduino IDE (1.8.5 Version)
  • ESP8266 and Blynk libraries
  • Blink Application

How it's working

  The Smart Helmet is Integrated with all the hardware Components which are provided in the list. The sensors sensed and provide the respective data and give it to the ESP8266 Microcontroller.
  The Microcontroller access the data and provides output signals according to the code which we had dumped into it. The sensed data is displayed in the Blink app and if any dangr conditions will arisen the Blink app givees alarams to alert the management and the management can save the workers.
